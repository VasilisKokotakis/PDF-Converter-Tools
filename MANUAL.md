# User Manual

## What is this?

Two simple desktop tools:

- **Image to PDF** – combine one or more photos into a single PDF file
- **PDF to Word** – extract the text from a PDF and save it as a Word document (.docx)

---

## How to open

Double-click **PDFConverterTools** to open the launcher, then click the tool you want.

---

## Image to PDF

### Basic steps

1. Click **Browse Files** to pick your images - or drag and drop them directly onto the list
2. Check the order - use the **up and down arrows** on each row to rearrange
3. Click **Convert to PDF**
4. Choose where to save the file
5. Done

### Tips

- You can add images from multiple folders - just click Browse Files again
- To remove a single image, click the **X** button on its row
- To start over, click **Clear All**
- Supported formats: JPG, PNG, BMP, TIFF, WEBP

---

## PDF to Word

### Basic steps

1. Click **Select PDF** and pick your PDF file
2. The file name, size, and page count will appear on screen
3. Click **Convert to Word**
4. Choose where to save the file
5. Done

### Things to know

- Only **text** is extracted - images, tables, and formatting inside the PDF are not carried over
- If the PDF is **password-protected**, the tool will tell you and stop - it cannot convert locked files
- To pick a different PDF, click **Clear** and then **Select PDF** again

---

## Keyboard shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + O` | Open / select files |
| `Ctrl + Enter` | Start conversion |
| `Escape` | Clear selection |

---

## Something went wrong?

- **"No images selected"** – click Browse Files or drag some images in before converting
- **"No file selected"** – click Select PDF first
- **"Password-protected"** – the PDF is locked, this tool cannot open it
- **Conversion failed** – the file may be corrupted or in an unsupported format; try a different file
